## Pré-requisitos

Antes de começar, verifique se você tem instalado:

- Node.js em uma versão compatível com Angular 21
- npm
- Git
- Angular CLI, caso queira usar comandos `ng` diretamente

## Instalando o projeto

Clone o repositório:

```bash
git clone <url-do-repositorio>
```

Acesse a pasta do projeto:

```bash
cd savio_tomaz
```

Instale as dependências:

```bash
npm install
```

## Usando o projeto

Para iniciar o servidor de desenvolvimento:

```bash
npm start
```

Depois, acesse:

```text
http://localhost:4200
```

Para gerar uma versão de produção:

```bash
npm run build
```

Para executar os testes:

```bash
npm test
```

Para executar o lint:

```bash
npm run lint
```

## Contribuindo

Este é um portfólio pessoal, mas sugestões e melhorias são bem-vindas.

Para contribuir:

1. Faça um fork deste repositório.
2. Crie um branch: `git checkout -b minha-melhoria`.
3. Faça suas alterações e confirme: `git commit -m "minha melhoria"`.
4. Envie para o branch: `git push origin minha-melhoria`.
5. Abra uma pull request.

Como alternativa, consulte a documentação do GitHub sobre [como criar uma pull request](https://docs.github.com/pt/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/creating-a-pull-request).
